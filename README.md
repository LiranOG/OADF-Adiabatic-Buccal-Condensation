## 1. Abstract

This repository hosts the formal treatment of the **Oral Adiabatic
Decompression Fog (OADF)** event: the transient, visible white plume
produced when a sealed human buccal cavity is mechanically pressurized by
voluntary musculature and then rapidly opened. The phenomenon is identified
as a low-cost, biologically self-sufficient analogue of the Wilson expansion
cloud chamber (Wilson, 1911).

The model treats three preconditions as required for the observed visible
plume under rapid decompression:

1. Near-saturated water-vapor humidity at body temperature
   (*T*<sub>0</sub> ≈ 310 K, RH ≈ 100 %).
2. Transient overpressure attainable by buccinator and diaphragm action,
   Δ*P* ∼ 5–15 kPa.
3. A high concentration of soluble bio-aerosol cloud condensation nuclei
   (CCN) generated by saliva atomization.

The model is derived from the Poisson adiabatic relation, the
Clausius–Clapeyron equation, and Köhler theory, yielding a closed-form
expression for the post-decompression saturation ratio *S*(*Π*), where
*Π* = *P*<sub>2</sub>/*P*<sub>1</sub> is the buccal compression ratio.
For the reference case *Π* = 1.10 the model predicts Δ*T* ≈ 8.4 K,
*S* ≈ 1.46 (46 % supersaturation), and an excess condensed water content of
≈ 12.9 g·m⁻³. The four-phase decomposition — Saturation, Compression,
Adiabatic Decompression, Condensation — is presented with explicit governing
equations.

OADF is positioned within the taxonomy of expansion-condensation phenomena
(Wilson chamber; champagne-cork plume; transonic Prandtl–Glauert
condensation; atmospheric cloud formation) and is mechanistically
distinguished from exhaled-breath mixing fog. No claim of physical novelty
is made; the contribution is terminological, pedagogical, and
aerosol-biological.

## 2. Reference Predictions

| Quantity                               | Value (Π = 1.10)               |
|----------------------------------------|--------------------------------|
| Initial temperature *T*<sub>0</sub>    | 310 K                          |
| Adiabatic index γ                      | 1.40                           |
| Temperature drop Δ*T*                  | 8.4 K                          |
| Saturation ratio *S*                   | 1.46                           |
| Supersaturation *s* = *S* − 1          | 0.46 (46 %)                    |
| Excess condensed water content         | 12.9 g·m⁻³                     |
| Decompression timescale τ<sub>exp</sub>| 30 µs (sonic) – 10 ms (vol.)   |

| Plume opacity onset                    | ~10 ms                         |
| Plume dissipation                      | ~1 s                           |

## 4. Build Instructions

### 4.1 Prerequisites

| Tool      | Minimum version                | Role                                |
|-----------|--------------------------------|-------------------------------------|
| `pdflatex`| TeX Live 2021 / pdfTeX 1.40    | LaTeX engine (mandatory)            |
| `latexmk` | 4.70                           | Driver (optional, preferred)        |
| `python3` | 3.9                            | HTML validator, local web server    |
| `make`    | GNU Make 3.81                  | Build orchestration                 |

Required LaTeX packages: `siunitx`, `mhchem`, `tikz`, `booktabs`,
`hyperref`, `natbib`, `epigraph`.
No external bibliography tool (`biber`, `bibtex`) is required; the
reference list is a self-contained `thebibliography` environment.

### 4.2 Compile the paper

```bash
make paper
```

`latexmk` is selected when available; otherwise a three-pass `pdflatex`
sequence is invoked. Output: `docs/paper/OADF_academic_paper.pdf`.

### 4.3 Serve the HTML articles locally

```bash
make web                # binds 0.0.0.0:8000
make web WEB_PORT=8080  # override port
```

### 4.4 Validate HTML structure

```bash
make validate
# or directly:
python3 scripts/validate_html.py docs/web
```

### 4.5 Clean

```bash
make clean       # transient LaTeX artifacts only
make distclean   # also remove the compiled PDF
```

### 4.6 Target enumeration

```bash
make help
```
